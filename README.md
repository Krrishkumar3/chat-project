# Chat Project

A real-time chat application built with WebSockets.

## 🚀 Live Demo

- **Frontend (Vercel):** [https://frontend-six-ebon-78.vercel.app](https://frontend-six-ebon-78.vercel.app)
- **Backend (Render):** [https://chat-project-d9v4.onrender.com](https://chat-project-d9v4.onrender.com)

## Tech Stack

- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **Backend:** Node.js, TypeScript, WebSocket (`ws`)

## Local Development

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

> Make sure to check the WebSocket port in the frontend config.
